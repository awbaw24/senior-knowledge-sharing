# 🏛️ 은퇴자 지식 공유 플랫폼 (Senior Knowledge Sharing)

## 🌟 프로젝트 비전

**"평생에 걸쳐 쌓아온 소중한 경험과 지혜를 다음 세대에게 전하고, 농어촌 활성화와 사회적 가치 창출에 기여하는 지식 공유 생태계"**

은퇴 전문가들의 귀중한 실무 경험과 노하우를 체계적으로 보존하고, 후배들과의 멘토링을 통해 지속가능한 지식 전수 문화를 만들어갑니다.

## 🚀 **실제 작동하는 웹사이트가 완성되었습니다!**

### 📱 웹사이트 기능

**🔗 [웹사이트 바로가기](https://awbaw24.github.io/senior-knowledge-sharing/website/)**

#### 💻 메인 기능
- **👥 멘토-멘티 매칭 시스템**: 실시간 등록 및 매칭
- **📚 지식 저장소**: 분야별 전문 지식 공유
- **🏘️ 지역별 커뮤니티**: 농어촌 활성화 프로젝트
- **📊 관리자 대시보드**: 실시간 현황 모니터링

#### 🎯 핵심 페이지
1. **메인 페이지** (`website/index.html`)
   - 히어로 섹션 및 통계
   - 분야별 전문가 카테고리
   - 성공 사례 및 지역별 커뮤니티

2. **관리자 대시보드** (`website/admin.html`)
   - 멘토/멘티 승인 관리
   - 매칭 현황 모니터링
   - 실시간 통계 및 알림

### 🛠️ 기술 스택
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **디자인**: 반응형 웹디자인, CSS Grid/Flexbox
- **아이콘**: Font Awesome 6.0
- **데이터**: Local Storage (데모용, 실제로는 백엔드 DB 연동)

### 📁 프로젝트 구조

```

│   ├── 🏗️ construction/            # 건축/토목 분야

    └── admin.js                    # 관리자 JavaScript
```

## 🎯 주요 목표

### 🔄 지식 순환 생태계 구축
- **은퇴 전문가 → 현직자**: 실무 경험과 노하우 전수
- **현직자 → 신입자**: 체계화된 지식의 재전달
- **세대 간 협력**: 전통 기법과 현대 기술의 융합

### 🌱 사회적 가치 창출
- **농어촌 활성화**: 6차 산업, 체험관광, 직거래 활성화
- **일자리 창출**: 은퇴자 창업 지원 및 컨설팅 기회 제공
- **지역 경제**: 소상공인 멘토링 및 지역 특화산업 육성

### 💡 지식 보존 및 혁신
- **무형 자산 보존**: 사라져가는 전통 기술과 노하우 문서화
- **실무 중심**: 이론이 아닌 현장에서 검증된 살아있는 지식
- **지속가능성**: 다음 세대가 계속 발전시킬 수 있는 지식 체계

## 🚀 웹사이트 사용법

### 👨‍🏫 멘토로 참여하기 (은퇴 전문가)

1. **웹사이트 접속**: [메인 페이지](https://awbaw24.github.io/senior-knowledge-sharing/website/)
2. **멘토 등록**: "🧑‍🏫 멘토로 참여" 버튼 클릭
3. **정보 입력**: 
   - 전문 분야 선택 (건축/토목, 제조업, 농업/어업, 서비스업)
   - 경력 및 전문성 기술
   - 연락처 정보
4. **승인 대기**: 관리자 검토 후 활동 시작

### 👨‍💼 멘티로 참여하기 (학습자)

1. **웹사이트 접속**: [메인 페이지](https://awbaw24.github.io/senior-knowledge-sharing/website/)
2. **멘티 등록**: "🎓 멘티로 시작" 버튼 클릭
3. **학습 목표 설정**: 관심 분야 및 학습 목표 명시
4. **멘토 매칭**: 시스템이 최적의 멘토 추천

### 🔍 멘토 찾기

1. **분야별 탐색**: 메인 페이지에서 관심 분야 카드 클릭
2. **멘토 목록 확인**: 해당 분야 전문가 리스트 확인
3. **멘토링 신청**: "💬 멘토링 신청" 버튼으로 연락

### 👨‍💻 관리자 기능

1. **관리자 접속**: [대시보드](https://awbaw24.github.io/senior-knowledge-sharing/website/admin.html)
2. **멘토 승인**: 신규 멘토 신청 검토 및 승인
3. **매칭 관리**: 멘토-멘티 최적 매칭 수행
4. **통계 모니터링**: 실시간 현황 및 성과 추적

## 💼 활용 사례

### 🏭 제조업 기술 전수
```
📈 성과: 30년 경력 기계가공 전문가 → 신입 엔지니어
- 정밀 가공 기법 전수로 불량률 50% 감소
- 설비 트러블슈팅 노하우로 가동률 20% 향상
- 품질관리 체계 구축으로 고객 만족도 95% 달성
```

### 🌾 농어촌 비즈니스 창업
```
📈 성과: 은퇴 농업인 → 귀농 청년
- 친환경 농법 지도로 유기농 인증 취득
- 직거래 노하우 전수로 소득 200% 증가  
- 농촌체험 프로그램 개발로 지역 관광 활성화
```

### 🏗️ 건축 실무 멘토링
```
📈 성과: 은퇴 건축사 → 신규 건축사
- 인허가 절차 완전 정복으로 업무 효율 300% 향상
- 현장 안전관리 노하우로 사고 제로 달성
- 고객 소통 기법으로 수주율 40% 증가
```

## 🤝 기여 방법

### 💡 지식 공유자 (멘토)
- **경험 사례 공유**: 성공/실패 사례를 솔직하게
- **실무 기법 전수**: 책에서 배울 수 없는 현장 노하우
- **문제 해결 지원**: 후배들의 궁금증과 어려움 해결
- **네트워킹 지원**: 업계 인맥과 정보 공유

### 📚 학습자 (멘티)
- **적극적 질문**: "왜"와 "어떻게"를 끊임없이 질문
- **실습 도전**: 이론을 실제로 적용해보기
- **피드백 공유**: 배운 내용과 개선사항 공유
- **확산 기여**: 동료들과 지식 나누기

### 🛠️ 개발자 기여
- **기능 개선**: 웹사이트 기능 추가 및 개선
- **버그 수정**: Issues 등록 및 Pull Request
- **문서화**: 사용법 및 개발 가이드 작성

## 🌍 사회적 임팩트

### 📊 기대 효과
- **지식 보존**: 매년 은퇴하는 수만 명의 전문가 지식 보존

- **SDG 11 (지속가능한 도시)**: 농어촌 지역 발전 기여
- **SDG 17 (파트너십)**: 세대 간 협력 플랫폼 구축
