## 👥 기여 방법

### 지식 공유자 (멘토)
1. **경험 공유**: 실무에서 겪은 사례와 해결 방법
2. **노하우 전수**: 도면이나 사진과 함께 상세한 설명
3. **질문 답변**: 후배들의 궁금증 해결

### 학습자 (멘티)
1. **적극적 질문**: 궁금한 점은 언제든 문의
2. **사례 요청**: 특정 상황에 대한 조언 요청
3. **학습 공유**: 배운 내용을 정리해서 공유
